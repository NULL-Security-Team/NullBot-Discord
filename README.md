# Nullbot-Reboot - [BETA] - Discord Auditing Bot

NullBot Beta is a multi-purpose Discord bot for system administrators and security enthusiasts. The bot is capable of executing commands through Discord to perform various tasks, such as detecting the operating system and finding open ports on a host using Nmap, and scanning a web server for vulnerabilities using in house custom scripts. The bot is also equipped with an authorization system to prevent unauthorized users from running certain commands. This bot was developed by [SirCryptic](https://github.com/sircryptic) of [NullSecurityTeam](https://github.com/orgs/NULL-Security-Team), and is open-source under the [MIT license](https://github.com/SirCryptic/Nullbot-Reboot/blob/main/LICENSE).
